Issue 1 — Task Types Distribution Chart Shrinks When Browser Extension Is Open
The Task Types Distribution donut chart shrinks and its legend text scales up disproportionately when a browser extension panel (e.g., AI Chat Assistant sidebar) is open on the right side, reducing the effective viewport width.
Fix Requirements:

Set a min-width (e.g., 280px) and a fixed height (e.g., 280px) on the Task Types Distribution chart container so it does not shrink below a usable size.
Add flex-shrink: 0 to the chart card/container so it resists compression in flex layouts.
Ensure the Chart.js canvas element has explicit width and height attributes, or is wrapped in a div with overflow: hidden and fixed dimensions.
Define legend text font size in px or rem (not em or %) so it does not scale relative to the shrunken container.
Optionally wrap the charts section in a min-width: 600px container with overflow-x: auto to prevent layout breakage on narrow viewports.

Issue 2 — GeniPilot Logo Redirects to Browser Home Page Instead of Dashboard
Clicking the GeniPilot logo in the sidebar navigates the user away to the browser's default home page instead of staying within the app.
Fix Requirements:

Locate the anchor tag wrapping the GeniPilot logo in the sidebar.
Change its href to point to the correct internal route (e.g., dashboard.html#overview or / depending on the routing setup).
If using a JavaScript router (e.g., SPA), replace the <a href> with an onclick handler that calls the internal navigation function instead of triggering a full page reload.
Ensure the logo never navigates outside the application under any circumstance.

Fix Requirements:

The voice should sound natural, clear, and pleasant — not robotic or monotone.
